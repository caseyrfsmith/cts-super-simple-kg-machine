# Getting Started with WeatherFlow API

Welcome to WeatherFlow! This guide will help you make your first API request.

## Prerequisites

Before you begin, you'll need:
- A WeatherFlow API key (sign up at our developer portal)
- Basic knowledge of REST APIs
- A tool like curl or Postman

## Quick Start

1. **Get your API key** - Sign up for a free developer account
2. **Authenticate** - Learn about [authentication](authentication.html)
3. **Make your first request** - Follow our [quickstart tutorial](quickstart-part-1.html)
